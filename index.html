How to use this Gradients

These Gradients are easy to use and will make your website more attentive and attractive. You can do these by following these steps:

  1. Write the Background-image property in the css of element.
  2. Choose linear-gradient() or radial-gradient().
  3. Click on Click to copy button on this website.
  4. Now, paste it in the ().

Also, there is a section in navbar named `Trial` . It is a live code editor. You can test this gradients there!
